# mentorhub-analytics
Проект-портфолио: аналитика платформы Mentorhub (SQL, Power BI, Python)

## О проекте

**Контекст:**  
Онлайн-сервис Mentorhub — платформа, которая помогает менторам и менти находить друг друга. Встречи проходят на площадке сервиса, назначить их можно в личном кабинете после авторизации на сайте.

**Задача:**  
Продакт-менеджер готовится к новому этапу развития приложения и запросил комплексный анализ данных о работе сервиса.

**Цель:**  
Предоставить данные для принятия решений по развитию продукта, выявить проблемные зоны и точки роста.

**Стек:**  
- **SQL** (PostgreSQL) — анализ данных, оконные функции, когорты
- **Python** (Pandas, Matplotlib) — прогнозирование потребности в менторах
- **Power BI** — интерактивный дашборд с ключевыми метриками

## Ключевые выводы

### 1. Динамика роста
- Платформа стабильно росла с 3–7 активных пользователей в начале 2021 до пика в августе 2022 (509 менторов, 883 менти).
- С января 2022 менти растут значительно быстрее менторов (соотношение выросло с 1.1 до 1.7).

### 2. Удержание пользователей
- Менторы возвращаются на платформу в среднем через 28 дней (медиана), менти — через 47 дней.
- Значительный разрыв между средним и медианой говорит о наличии "хвоста" пользователей, возвращающихся через 100+ дней.

### 3. Нагрузка на менторов
- Средняя нагрузка выросла с 0.25 до пика **0.54 сессий в неделю (август 2022)**.
- Топ-менторы (**сентябрь 2022**) работают в 6.8 раза выше среднего.

### 4. Пиковые нагрузки (сентябрь 2022)
- Четверг — самый загруженный день для 5 из 7 направлений (пик у "Python для аналитики данных" — 65 встреч).
- Технические работы и обновления рекомендуется планировать на вторник/среду.
